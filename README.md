# Checkout Loader Proof of Concept

-   `apps/checkout` - a mock version of the new Checkout app
-   `apps/checkoutjs` - new checkout loader script
-   `apps/website` - seller website mock

## Getting Started

```sh
npm ci
npm run dev
```

Open http://localhost:8000 in the browser
